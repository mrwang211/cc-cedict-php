# CC-CEDICT php

Parser and downloader for the CC-CEDICT chinese dictionary https://www.mdbg.net/chinese/dictionary?page=cedict

## Installation

You can install the package via composer:

```bash
composer require mrwang211/cc-cedict-php
```
